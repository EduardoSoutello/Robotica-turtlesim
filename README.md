# Robótica - Turtlesim

Exercícios de movimentação usando Turtlesim do ROS.

## Descrição

Scripts Python para controlar movimentos da tartaruga no simulador Turtlesim.

## Arquivos

- `move.py` - Movimentação básica
- `movexy.py` - Movimentação no plano XY
- `movey.py` - Movimentação no eixo Y

## Conceitos

- ROS Turtlesim
- Comandos de Velocidade
- Controle de Robôs Móveis
- Coordinate Systems

## Tecnologias

- Python
- ROS (Robot Operating System)
- Turtlesim
